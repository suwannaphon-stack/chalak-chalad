แอปตรวจฉลากโภชนาการด้วย AI สำหรับผู้บริโภค

| รายการ | ราคา |
|--------|------|
| Netlify Hosting | ฟรี |
| Google Gemini AI | ฟรี (250 ครั้ง/วัน) |
| Domain (.netlify.app) | ฟรี |

## 🚀 วิธี Deploy ทั้งหมด 4 ขั้นตอน (ละเอียดทุก step)

---

### ขั้นตอนที่ 1: สร้าง Gemini API Key (ฟรี ไม่ต้องบัตรเครดิต)

1. เปิดเบราว์เซอร์ ไปที่ **https://aistudio.google.com**
2. กด **"Sign in"** → Login ด้วย Google Account (Gmail) ของคุณ
3. เมื่อเข้าสู่ระบบแล้ว มองหาเมนูด้านซ้าย กดที่ **"Get API Key"**
4. กดปุ่ม **"Create API key"**
5. ระบบจะสร้าง API key ให้ หน้าตาจะขึ้นต้นด้วย `AIzaSy...`
6. **กดปุ่ม Copy** เพื่อคัดลอก API key
7. **วาง (Paste) ไว้ใน Notepad หรือแอปจดบันทึก** เก็บไว้ใช้ในขั้นตอนที่ 4

> ✅ ไม่ต้องใส่บัตรเครดิต ฟรีจริงๆ
> ✅ ถ้ามี Gmail อยู่แล้ว ทำได้ภายใน 1 นาที

---

### ขั้นตอนที่ 2: สมัคร GitHub และ Upload โค้ด

GitHub คือเว็บเก็บโค้ด เหมือน Google Drive แต่สำหรับโค้ดเว็บไซต์

#### 2.1 สมัคร GitHub (ทำครั้งเดียว)

1. เปิดเบราว์เซอร์ ไปที่ **https://github.com**
2. กดปุ่ม **"Sign up"** (มุมขวาบน)
3. กรอกข้อมูล:
   - **Email** → ใส่อีเมลของคุณ
   - **Password** → ตั้งรหัสผ่าน
   - **Username** → ตั้งชื่อผู้ใช้ภาษาอังกฤษ (เช่น `somchai123`)
4. ทำตามขั้นตอนยืนยันตัวตน (อาจมีให้แก้ปริศนา)
5. ไปเช็คอีเมล → กดลิงก์ยืนยัน
6. **สมัครเสร็จ!**

#### 2.2 สร้าง Repository (ที่เก็บโค้ด)

1. Login เข้า GitHub แล้ว
2. กดปุ่ม **"+"** ที่มุมขวาบน (ข้างรูปโปรไฟล์) → เลือก **"New repository"**
3. กรอกข้อมูล:
   - **Repository name** → พิมพ์ `chalak-chalad`
   - **Description** → พิมพ์ `แอปตรวจฉลาก GDA` (หรือเว้นว่างก็ได้)
   - เลือก ✅ **Public**
   - ⬜ **ไม่ต้องติ๊ก** Add a README file (สำคัญ! อย่าติ๊ก)
4. กดปุ่มสีเขียว **"Create repository"**
5. จะเห็นหน้า repository ใหม่ที่ว่างเปล่า

#### 2.3 แตกไฟล์ zip

1. หาไฟล์ `chalak-chalad-netlify.zip` ที่ดาวน์โหลดจาก Claude
2. แตกไฟล์:
   - **Windows:** คลิกขวาที่ไฟล์ zip → เลือก **"Extract All..."** → กด **"Extract"**
   - **Mac:** ดับเบิลคลิกที่ไฟล์ zip
3. จะได้โฟลเดอร์ `chalak-chalad-netlify/` เปิดเข้าไปข้างใน จะเห็น:
   ```
   📁 netlify/
   📁 public/
   📄 .gitignore
   📄 README.md
   📄 netlify.toml
   ```

#### 2.4 Upload ไฟล์ขึ้น GitHub

1. กลับมาที่หน้า GitHub repository ที่สร้างไว้ในข้อ 2.2
2. มองหาข้อความ **"uploading an existing file"** (เป็นลิงก์สีฟ้า) แล้วกดเข้าไป
   - ถ้าไม่เห็น → กดปุ่ม **"Add file"** (ด้านบน) → เลือก **"Upload files"**
3. เปิดโฟลเดอร์ `chalak-chalad-netlify/` ในคอมพิวเตอร์ของคุณ
4. **กด Ctrl+A** (เลือกไฟล์ทั้งหมดข้างใน) → **ลากทั้งหมดมาวาง** บนหน้า GitHub ที่เขียนว่า "Drag files here"

   ⚠️ **สำคัญมาก:** ลากไฟล์และโฟลเดอร์ **ข้างใน** ไม่ใช่ลากโฟลเดอร์ `chalak-chalad-netlify` ทั้งก้อน

5. รอจน GitHub แสดงรายชื่อไฟล์ทั้งหมด (อาจใช้เวลาสักครู่)
6. เลื่อนลงล่างสุดของหน้า
7. กดปุ่มสีเขียว **"Commit changes"**
8. **Upload เสร็จ!** ✅

#### 2.5 ตรวจสอบว่า Upload ถูกต้อง

หน้า repository ควรแสดงไฟล์แบบนี้:

```
📁 netlify/
📁 public/
📄 .gitignore
📄 README.md
📄 netlify.toml
```

- กดเข้า `netlify/` → `functions/` → ควรเห็น `analyze.js`
- กดเข้า `public/` → ควรเห็น `index.html`

> ❌ ถ้าเห็นแค่โฟลเดอร์ `chalak-chalad-netlify/` อันเดียว = ลากผิด
> ให้ลบแล้ว upload ใหม่ โดยลากไฟล์ **ข้างใน** โฟลเดอร์

### ขั้นตอนที่ 3: สร้างเว็บบน Netlify

#### 3.1 สมัคร Netlify (ทำครั้งเดียว)

1. ไปที่ **https://www.netlify.com**
2. กดปุ่ม **"Sign up"** (มุมขวาบน)
3. เลือก **"Sign up with GitHub"** (จะเชื่อมต่อกันอัตโนมัติ)
4. GitHub จะถามอนุญาต → กด **"Authorize"**
5. **สมัครเสร็จ!**

#### 3.2 สร้างเว็บจาก GitHub

1. อยู่ในหน้า Netlify Dashboard
2. กดปุ่ม **"Add new site"** → เลือก **"Import an existing project"**
3. เลือก **"GitHub"**
4. ถ้า GitHub ถามอนุญาตอีกครั้ง → กด **"Authorize"**
5. จะเห็นรายชื่อ repository → **กดเลือก `chalak-chalad`**
6. หน้าตั้งค่าจะขึ้นมา ให้ตั้งค่าดังนี้:
   - **Branch to deploy:** `main` (ไม่ต้องเปลี่ยน)
   - **Build command:** **ลบออกให้หมด เว้นว่าง** (ห้ามพิมพ์อะไร)
   - **Publish directory:** พิมพ์ `public`
7. กดปุ่ม **"Deploy site"**
8. **รอ 1-2 นาที** Netlify จะสร้างเว็บให้
9. เมื่อเสร็จ จะเห็นลิงก์เว็บ เช่น `https://random-name-12345.netlify.app`

> 💡 อยากเปลี่ยนชื่อเว็บ?
> ไปที่ **Site configuration** → **Site details** → **Change site name**
> เช่น เปลี่ยนเป็น `chalak-chalad` จะได้ลิงก์ `chalak-chalad.netlify.app`

---

### ขั้นตอนที่ 4: ใส่ API Key (ขั้นตอนสุดท้าย สำคัญมาก!)

⚠️ ถ้าไม่ทำขั้นตอนนี้ แอปจะเปิดได้แต่ตรวจฉลากไม่ได้

1. อยู่ในหน้า Netlify Dashboard ของเว็บที่สร้าง
2. กดเมนู **"Site configuration"** (แถบด้านบน)
3. เมนูด้านซ้าย → กดที่ **"Environment variables"**
4. กดปุ่ม **"Add a variable"**
5. กรอก:
   - **Key:** พิมพ์ `GEMINI_API_KEY` (ตัวใหญ่ทั้งหมด ตามนี้เป๊ะ)
   - **Values:** วาง (Ctrl+V) API key ที่เก็บไว้จากขั้นตอนที่ 1 (`AIzaSy...`)
6. กด **"Create variable"**
7. กลับไปที่เมนู **"Deploys"** (แถบด้านบน)
8. กดปุ่ม **"Trigger deploy"** → เลือก **"Deploy site"**
9. รอ 30 วินาที - 1 นาที
10. เมื่อสถานะขึ้น ✅ **"Published"** (สีเขียว)

### 🎉 เสร็จแล้ว! เว็บพร้อมใช้งาน!

- เปิดลิงก์เว็บของคุณ (เช่น `https://chalak-chalad.netlify.app`)
- แชร์ลิงก์ให้ผู้บริโภคผ่าน LINE / Facebook ได้เลย
- ทุกคนเปิดลิงก์ใช้ได้ทันที ไม่ต้องติดตั้งอะไร!
